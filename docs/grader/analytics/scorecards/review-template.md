# Regrading Template

## Review Record

- **Revision:** Record the immutable Git revision.
- **Review date:** Record ISO date.
- **Scope:** State reviewed paths, devices, and excluded surfaces.
- **Methods:** List static inspection, runtime journeys, network/storage checks, and test commands.

## Evidence Register

Record strengths and findings by stable ID using the ledger schema. Preserve old IDs, add new IDs, attach exact path/line or runtime evidence, record severity/confidence/status, and link resolution evidence.

## Blockers, Perspectives, And Roadmap Delta

List cap-triggering IDs and removal verification. Summarize Apple, SpaceX, xAI, and Google priorities without changing ledger facts. List new, closed, and reprioritized roadmap IDs; explain score delta.

## Approval Record

Record reviewer, date, unresolved runtime evidence, and whether the package is approved for planning.

## Regrading Checklist

Capture revision; refresh static/runtime evidence; preserve IDs; register new findings; attach resolution evidence; apply caps; verify arithmetic; refresh perspectives; refresh roadmap and dependency map; publish delta.
